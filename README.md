# Tab Manager

URLパターンルールによるタブの自動振り分け・自動クローズを行う Google Chrome 拡張機能。

## 機能

- **ドメイン部分一致によるタブの自動振り分け** — ドメイン名のキーワードでルールを設定。タブが開かれた瞬間に自動でウィンドウやタブグループへ移動する
- **複数パターン対応** — 1つのルールに複数のドメインパターンを登録でき、いずれかにマッチした場合に適用される
- **タブグループはウィンドウ単位で管理** — 同名グループが他のウィンドウにある場合は別ウィンドウに移動せず、そのウィンドウ内だけでグループ化する
- **自動クローズ** — 設定した時間が経過したタブを自動で閉じる。`chrome.alarms` を使用するためブラウザ再起動をまたいでも機能する
- **クローズ履歴と再オープン** — 自動クローズされたタブを一覧で確認し、ワンクリックで再オープンできる

## インストール（ビルド不要）

```bash
git clone https://github.com/juki-bo/tab-manager.git
```

1. `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」をオンにする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. クローンした `tab-manager/dist/` フォルダを選択

## 開発者向け

```bash
npm install
npm run dev   # 開発サーバー起動（変更のたびに chrome://extensions/ でリロード）
npm run build # dist/ を再生成
```

## ルールの設定例

| ドメインパターン | 説明 |
|---|---|
| `github.com` | GitHub のすべてのページ |
| `linear.app` | Linear のすべてのページ |
| `notion.so` | Notion のすべてのページ |

1つのルールに複数パターンを登録することで、まとめて同じグループに振り分けられる。

例: `github.com` と `gitlab.com` を同一の「コード」グループへ振り分ける。

### タブグループの動作

| 状況 | 動作 |
|---|---|
| 同じウィンドウに同名グループあり | そのグループに追加 |
| 別ウィンドウに同名グループあり | グループ化しない（タブはそのまま） |
| どこにも同名グループなし | 現在のウィンドウに新規作成 |
