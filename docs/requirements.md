# タブマネージャー Chrome拡張機能 要件定義書

## 概要

Google Chrome向けのタブ管理拡張機能。URLパターンルールによる自動振り分け、時間経過による自動クローズ、閉じたタブの履歴管理を提供する。

---

## 背景・課題

- 複数ウィンドウでタブを管理すると、acid tab のような拡張機能ではウィンドウごとにタブ一覧が重複表示され煩雑になる
- タブの増殖を自動的にコントロールする仕組みがない
- 閉じてしまったタブを後から確認・復元したい

---

## 機能要件

### 1. URLパターンによるタブ自動振り分け

- URLパターン（glob または 正規表現）でルールを設定できる
- 新しいタブが開かれたとき、またはURLが変わったとき、ルールを評価して自動でウィンドウ／タブグループへ移動する
- ルールは複数設定でき、優先順位（priority）を持つ
- 各ルールに以下を設定できる
  - パターン文字列（glob / 正規表現）
  - 移動先ウィンドウID
  - 移動先タブグループ名・カラー
  - タブグループへの自動割り当て
  - 個別の自動クローズ時間（グローバル設定を上書き可能）
  - ルールの有効／無効切り替え

### 2. 時間経過による自動クローズ

- グローバルで自動クローズまでの時間（分）を設定できる（0 = 無効）
- ルール単位でも個別に自動クローズ時間を設定できる（グローバル設定を上書き）
- タブが開かれてから、またはURLが変わってから指定時間経過後に自動でタブを閉じる
- 閉じる前にタブ情報（URL・タイトル）を履歴に記録する
- ブラウザ再起動をまたいでもタイマーが機能する（chrome.alarms を使用）

### 3. 設定画面

- 拡張機能アイコンをクリックするとポップアップが開く
- ポップアップは最小限の情報のみ表示（タブ一覧は表示しない）
  - 有効なルール数
  - グローバル自動クローズ設定の状態
  - 設定ページへのリンク
- 設定ページ（別タブで開く）は以下の3セクションで構成
  - **ルール管理**
  - **グローバル設定**
  - **履歴**

### 4. 自動クローズ履歴・再オープン

- 自動クローズされたタブは履歴として保存される
- 設定画面の履歴セクションで以下が可能
  - 閉じたタブの一覧表示（ファビコン・タイトル・URL・閉じた日時・適用ルール名）
  - タブの再オープン
  - ページ単位の一括再オープン
  - タイトル・URLによる検索・フィルタ
- 履歴の最大保存件数はグローバル設定で変更可能（デフォルト200件）
- 「履歴をすべて削除」機能あり

### 5. タブ一覧の非表示

- 拡張機能のUI内でオープン中のタブ一覧を表示しない
- 複数ウィンドウを使用しても拡張機能のUIが増殖・重複しない
- ルール設定画面でウィンドウを選択する際は、ウィンドウID/ラベルのみ表示（タブ内容は表示しない）

---

## 非機能要件

### パフォーマンス

- タブ作成・URL変更イベントへの応答は 200ms 以内
- バックグラウンドサービスワーカーはイベント駆動で動作し、常時起動しない

### データ永続化

- すべての設定・履歴は `chrome.storage.local` に保存する
- ブラウザ再起動・拡張機能の再読み込み後もデータが維持される
- 自動クローズタイマーは `chrome.alarms` を使用してブラウザ再起動をまたいで維持する

### セキュリティ

- 必要最低限のパーミッションのみ要求する
  - `tabs`, `tabGroups`, `storage`, `alarms`, `windows`
  - `host_permissions: <all_urls>`（URLパターンマッチングに必要）

### 互換性

- Manifest V3 準拠
- Chrome 最新安定版（および過去2バージョン）をサポート

## ストレージスキーマ

```typescript
interface SortRule {
  id: string;
  enabled: boolean;
  name: string;
  pattern: string;
  patternType: "regex" | "glob";
  priority: number;
  targetWindowId?: number;
  targetGroupName?: string;
  targetGroupColor?: string;
  autoCloseMinutes?: number;
}

interface GlobalSettings {
  autoCloseMinutes: number;
  maxHistoryEntries: number;
}

interface ClosedTabRecord {
  id: string;
  url: string;
  title: string;
  faviconUrl?: string;
  closedAt: number;
  closedByRuleId?: string;
}
```

## 制約・スコープ外

- Firefox など Chrome 以外のブラウザはサポートしない
- タブの「ピン留め」状態の管理はスコープ外
- タブのスクリーンショット保存はスコープ外
- クラウド同期（`chrome.storage.sync`）はスコープ外（容量制限のため）
